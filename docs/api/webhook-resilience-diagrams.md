State Behaviors:

CLOSED (Normal):
  - All webhook calls go through
  - Track failure count
  - Open after 5 consecutive failures

OPEN (Fallback):
  - No webhook calls (immediate fallback)
  - Wait 60 seconds
  - Transition to HALF-OPEN

HALF-OPEN (Testing):
  - Allow test webhook calls
  - Track success count
  - Close after 3 successes
  - Reopen on any failure
